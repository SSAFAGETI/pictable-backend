해석 및 발표 포인트

  • 데스크톱 최종 성능은 99점, 모바일 뷰 최종 성능은 94점으로 확인했다.
  • 접근성 개선 검증에서 데스크톱과 모바일 모두 100점을 달성했다.
  • Unit, Browser Component, E2E 테스트를 계층화해 UI 단위와 실제 사용자 흐름을 함께 검증했다.

별도 네이티브 앱 빌드는 없으므로, 앱 항목은 모바일 사용자 환경을 대표하는 Lighthouse mobile emulation 결과로 정리했다.
