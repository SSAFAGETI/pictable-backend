프론트엔드 구조

Vue 3 SPA

Vite 기반 단일 페이지 앱이며 Vue Router로 화면 전환을 관리한다.

Feature Modules

인증, 레시피, 피드, 사용자, 추천 기능을 기능 단위로 분리한다.

API Client

apiRequest를 중심으로 백엔드 API와 미디어 요청을 통합한다.

Auth State

로그인 상태와 토큰 저장소를 관리하며 API 요청에 인증 정보를 연결한다.

주요 사용자 흐름

흐름화면 / 라우트연동
탐색/, /feed, /recipe/:id홈 요약, 피드, 레시피 상세 API
추천/ingredients, /recommendations미디어 업로드, 재료 감지, 추천 API
인증/login, /oauth/callbackGoogle OAuth, /auth/google/
마이페이지/mypage저장/좋아요/작성 레시피, 구독 관계

외부 서비스 분리

  • Food Safety Korea API는 프론트 API Client에서 직접 공공 레시피 이미지 조회에 사용한다.
  • Google OAuth와 Gemini 기반 재료 감지는 백엔드 중심으로 처리해 인증/AI 호출 책임을 분리한다.
  • Vercel Rewrite는 백엔드 API, 미디어, Django Admin 요청을 백엔드로 전달한다.
